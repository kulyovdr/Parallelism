Make: для выбора типа double собираем с помощью команды "make";
для выбора типа float собираем с помощью команды "make float".

CMake: для выбора типа double в файле CMakeLists.txt в последней команде приравниваем FL к любому числу, кроме 1;
для выбора типа float в файле CMakeLists.txt в последней команде приравниваем FL к 1.

Task1Double Make выводит 0.000000000157555771921765553701

Task1Float Make выводит 0.083180807530879974365234375000

Task1Double CMake выводит 0.000000000971553533911664147582

Task1Float CMake выводит 0.116326704621315002441406250000
